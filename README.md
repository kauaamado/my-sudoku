# KSudoku

[Português](#português) | [English](#english)

## Português

### Sobre o Projeto
O **KSudoku** é um jogo de Sudoku desenvolvido em Flutter. Este projeto foi criado como um método de estudo para aprofundar conhecimentos no framework Flutter e em desenvolvimento mobile/multiplataforma.

### Localização
O jogo está atualmente em Português devido à língua nativa do autor e ao contexto de desenvolvimento inicial.

### Desenvolvimento Assistido por IA
Este projeto contém trechos de código e lógicas que foram desenvolvidos com o auxílio de ferramentas de Inteligência Artificial, visando produtividade e aprendizado.

## English

### About the Project
**KSudoku** is a Sudoku game built with Flutter. This project was created as a study method to deepen knowledge in the Flutter framework and mobile/cross-platform development.

### Localization
The game is currently in Portuguese due to the author's native language and the initial development context.

### AI-Assisted Development
This project contains code snippets and logic developed with the assistance of Artificial Intelligence tools, focusing on productivity and learning.
